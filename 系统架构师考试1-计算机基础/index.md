# 数据库

## Armstrong公理系统
- 自反律：Y是X的子集，则X->Y
- 增广律：X->Y，则XZ->YZ
- 传递律：X->Y，Y->Z，则X->Z
- 合并：由X->Y，X->Z，有 X ->YZ （增广+传递）
- 伪传递：由X->Y，WY->Z，有WX->Z （增广+传递）
- 分解：由X->Y及Z⊆Y，有X->Z （自反+传递）
